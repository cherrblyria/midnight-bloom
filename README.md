- ColorScheme: **Catppuccin Mocha**
- Compositor: **Hyprland**

- Terminal: **Foot**
- File Manager: **Yazi**

### How to get this?

Basically just clone the repo, stow configs then pray..

```bash
git clone https://github.com/cherrblyria/midnight-bloom.git ~/.dotfiles
cd ~/.dotfiles
stow --no-folding --adopt yuri
```

#### Branches

- `main`: The default branch, contains the latest stable version.
- `lab`: The lab branch, contains experimental features.
